# 📊 Sales Analysis Power BI Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard designed for sales analysis. It provides meaningful insights into sales performance, trends, and key business metrics to support data-driven decision-making.

## 🎯 Objective
To analyze sales data and visualize important KPIs such as total sales, profit, region-wise performance, and product-wise insights using Power BI.

## 📈 Features
- Interactive dashboards with filters and slicers
- Region-wise and product-wise sales analysis
- Key performance indicators (KPIs)
- Trend analysis for better decision-making
- User-friendly and dynamic visualizations

- Power BI
- Data Visualization Techniques
